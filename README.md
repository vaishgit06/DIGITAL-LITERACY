# DIGITAL-LITERACY
This project consists of different tasks from Digital Literacy.

Task Summary
Task 1 – Digital Literacy Infographic

Created an infographic using Canva explaining digital literacy, safe internet practices, and useful digital tools for students.

Task 2 – Student Digital Portfolio

Created professional profiles on GitHub, LinkedIn, and Kaggle to build an online presence useful for academic and career growth.

Task 3 – Coding & Collaboration Platforms

Completed a beginner coding challenge on HackerRank and created a Google Form quiz to spread awareness about digital literacy.

Task 4 – Email Etiquette

Prepared a phishing cybercrime case study and created a prevention checklist to promote safe online practices among students.

Learning Outcome

Through this project, I learned the importance of digital literacy, online safety, professional communication, and maintaining a responsible digital identity. These skills will help me throughout my academic journey and future career.
